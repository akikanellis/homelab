## Prerequisite Software

Make sure you have the following installed:

* [Just](https://github.com/casey/just#installation)

## Useful Commands

1. List all commands

    ```shell
    just
    ```

2. Install project dependencies

    ```shell
    just install
    ```

3. Lint the project

    ```shell
    just lint
    ```

4. Run the tests

    ```shell
    just test
    ```

5. Run the app

    ```shell
    just run
    ```

6. Clean generated files

    ```shell
    just clean
    ```
